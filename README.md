# mincy

Svelte CLI

## Install

```
$ npm i -g mincy
```

## Usage

### Commands

```
# clone Svelte template
$ mincy clone sveltejs/template /projects/
```

```
# run
$ mincy run sveltejs/template /projects/
```

```
# start dev server
$ mincy start sveltejs/template /projects/
```

```
# build
$ mincy build sveltejs/template /projects/
```
